Ачивки:
I. 🚀 Прогресс и завершение
1️⃣ First Steps
Выдается:
→ за выполнение первой задачи (с программы обучения, либо отдельной задачи – без разницы)
2️⃣ Module Conqueror
→ завершение модуля без проваленных сабмитов
3️⃣ Boss Slayer 
→ завершение Boss Challenge

II. ⚡ Эффективность
(эти ачивки даются именно в отдельных задачах, не в тех которые с уроков программы обучения)
Speed Coder
→ задача решена быстрее среднего времени
Clean Code
→ код прошел все тесты и AI оценил стиль как “clean”
Zero Hint
→ решение без использования AI и Hint

III. 🎓 Уровневые ачивки (по треку)
С учетом USER FLOW 
Beginner Graduate
→ завершение Beginner Track
Intermediate Engineer
→ завершение Intermediate Track
Advanced Architect
→ завершение Advanced Track


IV. 🧩 За тип задач (даются за отдельные задачи)
Loop Master
→ 10 задач с циклами
Function Expert
→ 10 задач по функциям
OOP Builder
→ 5 задач с классами
Data Structures Ninja
→ 10 задач со словарями / списками

V. 👨‍🏫 Teacher-Based Achievements
Преподаватель может:
•	Добавить кастомную ачивку к лекции
•	Добавить ачивку к задаче
Например: 
Creative Solution
→ преподаватель вручную наградил за нестандартный подход
Best in Class
→ лучший результат в группе
(Это создаёт геймификацию внутри группы)
